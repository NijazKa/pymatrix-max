# Журнал изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
а версии следуют Semantic Versioning.

## [0.12.0] — 2026-07-17

Первый публичный preview-релиз.

### Добавлено

- многопользовательские MAX-сессии, изолированные по Matrix-пользователю;
- стабильный профиль Android-устройства для каждой MAX-сессии;
- авторизация по телефону, коду и паролю двухфакторной защиты;
- уведомления об истечении login token и диагностика reconnect;
- личные чаты, группы и каналы;
- создание DM по номеру телефона;
- вступление в группу/канал по invite-ссылке;
- выход, локальная блокировка и автоматический выход при повторном добавлении;
- таблица владельцев общих групповых portal и ручная команда `syncgroups`;
- relay-режим групп через bridge bot без массовой регистрации ghost-пользователей;
- текст, фото, видео, аудио, голосовые и файлы в обе стороны;
- сборка входящего MPEG-DASH-видео в MP4 через FFmpeg;
- двусторонние ответы с таблицей соответствий MAX message ID ↔ Matrix event ID;
- Matrix → MAX реакции и экспериментальная обработка MAX → Matrix;
- телефон контакта в теме личной комнаты и команда `contact`;
- локальный инструмент аудита неиспользуемых Synapse ghost-аккаунтов.

### Известные ограничения

- MAX → Matrix реакции могут не приходить из-за отсутствия push-события;
- не поддерживаются редактирование, удаление и backfill истории;
- нет E2EE и double puppeting;
- inline keyboard и часть служебных вложений пропускаются.
