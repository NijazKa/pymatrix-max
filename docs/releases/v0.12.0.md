# pymatrix-max v0.12.0

Первый публичный preview-релиз неофициального многопользовательского моста
Matrix ↔ MAX.

## Главное

- сообщения и медиа в обе стороны;
- двусторонние ответы;
- группы и каналы с relay через bridge bot;
- управление чатами: join, leave, block;
- DM по номеру и отображение доступного телефона контакта;
- устойчивые MAX-сессии и диагностика reconnect;
- MPEG-DASH видео автоматически собирается в MP4;
- Docker и systemd документация;
- MIT license.

## Перед обновлением

Сделайте резервную копию каталога `data/`. Не запускайте две копии моста с одним
каталогом MAX-сессий.

## Известные ограничения

MAX → Matrix реакции остаются экспериментальными. E2EE, backfill, edits/deletes и
double puppeting пока не поддерживаются.
