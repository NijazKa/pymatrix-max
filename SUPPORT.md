# Поддержка

- Ошибка моста: создайте Bug report в GitHub Issues.
- Предложение функции: создайте Feature request.
- Уязвимость: следуйте `SECURITY.md`.
- Вопрос эксплуатации: приложите версию, способ установки, релевантный фрагмент
  лога и направление сообщения.

Перед публикацией лога удалите телефоны, Matrix ID, room ID, event ID, token,
тексты личных сообщений и подписанные MAX CDN URL.
